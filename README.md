## 技術規範
- [Tailwind.css](https://tailwindcss.com/)
- [CSS Paged Media](https://www.w3.org/TR/css-page-3/)

## 執行專案

#### 方法ㄧ
打開目錄資料夾，對 index.html 點擊右鍵選擇開啟方式，選擇瀏覽器打開。

#### 方法二
下載並使用 VS Code 打開目錄，下載擴充套件 Live Server 執行 index.html。
